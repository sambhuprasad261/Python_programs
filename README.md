# Python Programs - Learning Python Basics

A collection of Python programs designed to help beginners learn Python programming fundamentals.

This repository contains Python programs covering the following topics:

### 1. Basics
- **hello_world.py** - Your first Python program
- **variables_datatypes.py** - Understanding variables and data types
- **operators.py** - Arithmetic, comparison, and logical operators

### 2. Control Flow
- **conditionals.py** - If/else statements and decision making
- **loops.py** - For and while loops with examples

### 3. Functions and Data Structures
- **functions.py** - Defining and using functions
- **lists.py** - List operations and methods
- **dictionaries.py** - Dictionary operations and methods
- **strings.py** - String manipulation and methods

### 4. File Operations
- **file_operations.py** - Reading and writing files

### 5. Mini Projects
- **calculator.py** - Simple calculator application

## üöÄ Getting Started

### Prerequisites
- Python 3.x installed on your system

### Running the Programs

Navigate to the directory and run any program:
```bash
python3 hello_world.py
```

Or:
```bash
python hello_world.py
```

## üìñ Learning Path

Follow the programs in this order for a structured learning experience:
1. Start with `hello_world.py`
2. Move to `variables_datatypes.py`
3. Continue with `operators.py`
4. Practice with `conditionals.py` and `loops.py`
5. Learn `functions.py`
6. Explore `lists.py`, `dictionaries.py`, and `strings.py`
7. Try `file_operations.py`
8. Build the `calculator.py` project

## üí° Tips
